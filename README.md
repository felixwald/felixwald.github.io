# Felix Wald — personal academic website

This repository contains the source for Felix Wald's personal website and online CV, built with Jekyll and the Academic Pages theme.

## What is included

- Homepage and professional summary
- Online CV
- Publications page with external profile links
- Project highlights / portfolio
- Contact information and downloadable CV asset

## Local preview

To preview the site locally:

```bash
bundle install
bundle exec jekyll serve -l -H localhost
```

Then open <http://localhost:4000>.

## Build

```bash
bundle exec jekyll build
```

## Deployment

The public GitHub Pages site should only be updated after review and approval by merging the approved branch to the publishing branch.
